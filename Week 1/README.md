# Week 1 — Python Sprint & Git Foundation

This week focuses on **Python fundamentals, modular programming, object-oriented programming, and professional Git & GitHub workflow**.  
All tasks are implemented using Python `.py` files and pushed to GitHub.


---

## Task Overview

### Task 1 — Python Basics & Git
- Python fundamentals: variables, loops, conditionals, functions
- Simple Python programs: `area_of_circle`, `factorial`, `string_reverse`, `number_guessing_game`
- Git & GitHub professional workflow

### Task 2 — Modular Python, Data Handling & Exceptions
- Advanced Python functions: `*args`, `**kwargs`, `lambda`
- Reusable utility functions and modular code
- File handling: read, write, append
- Exception handling: try/except/else/finally
- Built-in data structures and comprehensions

### Task 3 — Object-Oriented & Advanced Python
- Object-Oriented Programming: classes, objects, encapsulation, inheritance, polymorphism
- Magic methods (`__init__`, `__str__`)
- Decorators for execution time measurement
- Iterators & generators (Fibonacci, custom range)
- Debugging awareness
